# Resume-Builder
A dynamic and interactive Resume Builder Web Application that allows users to create, edit, and preview a professional resume in real-time.

Overview
This project is designed to simulate a real-world resume creation tool where users can input their details and instantly see a formatted resume preview.

The goal of this project is to strengthen frontend fundamentals, DOM manipulation, and structured data handling using JavaScript.
🎯 Features
✅ Implemented
📌 Add & Edit Contact Information
📌 Professional Summary Section
📌 Dynamic Experience Section (Add / Delete / Edit)
📌 Education History Management
📌 Honors & Awards Section
📌 Skills Section (comma-separated → structured list)
📌 Live Resume Preview (Modal-based UI)

🚧 In Progress
Responsive Design (Mobile-friendly layout)
Improved UI/UX (Modern styling & spacing)
Form Validation (Better user input handling)
Export Resume as PDF
Local Storage / Data Persistence
